# Aperçu du Logo Finalisé GPC / GPS

Ouvrez ce fichier dans l'éditeur de Markdown de votre IDE pour voir le rendu :

![Aperçu du Logo GPC](./public/logo_gpc_preview.png)

Ce logo combine :
- Un **G** (dessiné sous forme d'arc de cercle rouge parfait avec une barre horizontale et verticale entrante).
- Des **reflets argentés** incrustés.
- Une **étoile dorée à quatre branches** (GPS Spark) parfaitement centrée.
- Aucun fond ni ombre portée, offrant une transparence parfaite sur le site.
